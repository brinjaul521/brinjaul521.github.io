一句话结论
“源码”和“生成的网页”是两件东西，要分开管：

  1. 源码（markdown、主题、配置）→ 用 git push 推到 源码仓库（可选，但强烈建议）。
  2. 生成的静态文件（public 目录）→ 用 hexo d 推到 GitHub Pages 仓库（brinjaul521.github.io）。

总结
